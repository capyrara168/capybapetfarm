## 🇬🇧 English

This is a professional business landing page for **Capyba Pet Farm**, an interactive animal park in Skudai, Johor.

### 🌟 Key Features
- **Calico Cat Theme**: A warm, cute, and family-friendly design inspired by the colors of a calico cat (Cream, Orange, and Dark Coffee).
- **Multilingual Support**: Fully localized in English, Chinese, and Malay with a seamless toggle system.
- **Mobile Optimized**: Featuring a horizontal "Animal Friends" scroll gallery specifically designed for mobile users.
- **Interactive Elements**: Dynamic Menu viewer with zoom capability and integrated Google Maps.

### 🛠️ Tech Stack
- HTML5, CSS3, JavaScript (Vanilla)
- Hosted on GitHub Pages

## 🇲🇾 Bahasa Melayu (Malay)

Ini ialah laman web perniagaan rasmi untuk **Capyba Pet Farm**, sebuah taman haiwan interaktif yang terletak di Skudai, Johor.

### 🌟 Ciri-ciri Utama
- **Tema Kucing Calico**: Rekaan yang mesra keluarga dan comel menggunakan palet warna putih krim, oren, dan kopi gelap.
- **Sokongan Pelbagai Bahasa**: Sistem penukaran bahasa antara Bahasa Melayu, Inggeris, dan Cina secara masa nyata.
- **Optimumkan untuk Mudah Alih**: Galeri "Kawan Haiwan" jenis leret sisi yang direka khas untuk pengguna telefon pintar.
- **Integrasi Penuh**: Paparan menu makanan boleh zum, pautan WhatsApp, dan lokasi Google Maps.
